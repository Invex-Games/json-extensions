Class JsonUtil

Provides extension members on JsonObject for flattening a hierarchical JSON structure into a single level and reconstructing (unflattening) it again.

Path conventions used by this class:

  • Object property segments are joined with a configurable separator (default ":"), e.g., "user:address:city".
  • Array elements are addressed with bare zero-based numeric segments, e.g., "users:0:name".

Note: this differs from Flatten(JsonNode, string), which uses bracketed indices such as "users:[0]:name".
